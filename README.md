# Sample App

## Live Deployments

| | URL |
|---|---|
| **Frontend** | https://sample-app-frontend.vercel.app |
| **Backend** | https://sample-app-backend-teal.vercel.app |

## Tech Stack

- **Frontend** — React (Create React App), React Router
- **Backend** — Flask (Python), PostgreSQL (Supabase)
- **Hosting** — Vercel

## Local Development

**Backend**
```bash
cd BackendAPI
pip install -r requirements.txt
python app.py
```

**Frontend**
```bash
cd Frontend
npm install
npm start
```

Frontend runs on http://localhost:3000, backend on http://localhost:5000.
